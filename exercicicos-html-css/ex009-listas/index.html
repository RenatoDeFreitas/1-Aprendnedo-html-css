Listas Ordenadas

Nesse tipo de lista, a ordem importa. Usaremos as TAG's "ol" encapsulando a "li". Neste modelo o HTML já insere os númerio. Para mudar a formatação usa-se "type" e para começar em otro valor usa-se o "style = 4", assim ela começará no número 4. Veja, abaixo:

  1. acordar
  2. tomar café
  3. arrumar a cama
  4. Ler notícias
  5. ir a academia
  6. iniciar trabalho
  7. Almoço
  8. Começar a estudar

Listas Não Ordenadas

Esse tipo de lista a sequência não guarda importância, então não precisa de marcação. A diferença é que inicia com a tag "ul" prescindida da tag "li". Nesta lista podemos personalizar o style usando os código de square, disc, circle.

  • pão
  • leite
  • açúcar
  • manteiga
  • arroz
  • feijão

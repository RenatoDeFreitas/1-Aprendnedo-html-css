Inserindo vídeo local

O vídeo a seguir está alocado em meu servidor. Atenção, para vídeo deve ser inserido por meio de "source' utilizando os formatos de "mp4, webm e ogg", todos usando usando os recursos (source) para abrir caso um formato não seja compatível com determinada página do navegador.

Além disso, é interessante que use o comando "poster" para mostrar o frame inicial.

Otimizando os custos

Para otimizar os custos de hospedagem, basta usar uma plataforma de hospedamgem de vídeo, como o youtube, por exemplo.

Inserindo vídios do Youtube

Quando está assistindo o vídeo pelo youtube não está consumindo direto do servidor, mas rodando pelo youbute. A única questão é que o vídeo está em aberto.

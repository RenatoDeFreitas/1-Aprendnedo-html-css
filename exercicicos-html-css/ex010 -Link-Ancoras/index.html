Usando Links externo. 🔗

O link externo via lhe direcionar para outra página web. A exemplo voce poderá acessar o meu Linkedin para conhecer um pouco mais do meu perfil profissional. Para isso foi necessário envelopar usando a tag "a" de âncora, a seguir é inserido o endereço da url completa que se deseja inserir.

Para ter acesso aos meus projetos é possível acessar o meu repositorio GitHub. Nesta página tem os projetos que estou desenvolvendo ou desenvolvi.

Lembre-se de usar o "target ="blank" rel= "external" para manter sua página em aberto!

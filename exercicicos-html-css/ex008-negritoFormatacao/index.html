Principais formatações 🔥

Negrito / Destaque

Há duas formas de negrito. Maneira semântica a "tag b" de boolt. Ela é uma tag sem sentido.

Agora, nesta frase, o termo em o negrito está inserido como destaque usando a tag "strong". O strong além da forma é semântico.

Itálico 🢄

Para inseir o termo em itálo, é necessário utilizar a tag "i" (não semântico.

Essa frase que acabou de ler temos um termo em ênfase usando a tag "EM"(semântica).

Marcador de texto

Também é possível usar um texto marcado uasndo a tag "MARK.

Já neste parágrafo temos outro texto marcado no final.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Aqui iremos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

Também existe o texto sublinhado com a tag "u" não semântica.

Texto sobrescrito

Esse texto usa-se para inserir aquelas letrinha pequenas em cima tipo... X25 + 3

Texto subscrito

Esse tipo de texto é para as letras pequenas em baico, tipo... H2SO4
